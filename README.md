# Getting started & Windows

### Dependencies
+ spring-boot-starter-web
+ spring-boot-starter-data-jpa
+ lombok
+ mysql-connector-j

### Requirements
 + java 17
 + gradle with java 17

### Installation and start

1. Clone the repo
```sh
git clone https://github.com/DanikMotolyha/digitalchief.git
```
2. go to the project
```sh
cd digitalchief
```
3. use gradle to start Spring Boot Application
```sh
gradle bootRun
```
